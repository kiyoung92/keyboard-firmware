## Installation
Use [QMK Toolbox](https://qmk.fm/toolbox) to flash the firmware after putting your keyboard into Reset mode.

## Usage
### OS Switching (Windows / Mac)

**To Mac**: Press FN1 + Left Alt to switch to the Mac (Layer 1) layout.

**To Windows**: Press FN1 + Left Command to switch back to the Windows (Layer 0) layout.

### Windows Key Lock
**Note**: This feature is only available when the Windows (Layer 0) layout is active.

**Enable Lock**: Press and hold FN1 + Left Windows for 3 seconds to disable the Windows key.

**Disable Lock**: Press and hold FN1 + Left Windows again for 3 seconds to re-enable the Windows key.
